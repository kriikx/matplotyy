# 📊 Matplotlib & Seaborn

This repository contains my learning notes, practice programs, and examples for **Matplotlib** and **Seaborn**, two of the most popular Python libraries for data visualization.

The notebooks were created while learning these libraries and cover essential concepts required for **Data Analysis, Exploratory Data Analysis (EDA), Machine Learning, and placements**.
